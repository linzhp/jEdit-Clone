Chapter 5. Advanced Text Editing

Working With Paragraphs

As far as jEdit is concerned, a "paragraph" is a block of text enclosed with double-newlines. This is also how TeX defines a paragraph. Note that jEdit doesn't parse HTML files for "<P>" tags, nor does it support paragraphs delimited only by a leading indent.

Control-Up moves the caret to the previous paragraph. Control-Down moves the caret to the next paragraph. Holding down shift in addition to one of these keys will extend the selection, instead of moving the caret.

Edit>Selection>Select Paragraph (keyboard equivalent: Control-E P) will select the paragraph containing the caret.

Edit>Paragraphs>Delete Paragraph (keyboard equivalent: Control-E Control-D) will delete the paragraph containing the caret.
