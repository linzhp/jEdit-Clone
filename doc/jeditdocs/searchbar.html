Chapter 7. Search and Replace

The Search Bar

The "search bar" provides quick access to searching functions. To search for a string, enter it into the search bar and press Enter. If Enter is pressed while no string is entered, the Search and Replace dialog box will be displayed. Pressing Escape will move the focus back to the main text area.

Search>Toolbar Find (keyboard equivalent: F7) will insert the currently selected text into the search bar and move keyboard focus to it. The search bar stores previously entered strings; see the section called History Text Fields.

All checkboxes to the right of the text field should be self-explanatory except Incremental, which activates incremental search mode, which is only available in the search bar. In incremental search mode, the search bar searches for the string immediately while you enter characters, instead of only searching when Enter is pressed.

Search>Toolbar Incremental Search (keyboard equivalent: F8) is like the Toolbar Find command except that it selects the Incremental checkbox for you. The search bar stores previously entered strings; see the section called History Text Fields.

If you don't like the search bar, it can be disabled in the Utilities>Global Options dialog box; see the section called The Global Options Dialog Box in Chapter 8 for details.
