Chapter 5. Advanced Text Editing

Rectangular Selection

Using jEdit's rectangular selection features, you can select and operate on columns of text.

There are three ways to create a rectangular selection. The simplest way is to drag the mouse from one corner of the rectangle to the other while holding down the Control key.

Clicking the mouse button while holding down Shift-Control will create a rectangle from the caret position to where the mouse was clicked.

Edit>Selection>Rectangular Selection (keyboard equivalent: Control-\) will toggle the current selection between continuous and rectangular mode.

It is possible to select a rectangle with zero width. For example, this can be used to insert a new column between two existing columns. Such "zero-width" selections are shown as a thin vertical line.

Deleting, replacing, copying, pasting, and so on works as expected on rectangles. Some plugins might not be rectangle-aware, and cause problems. All commands built in to jEdit should work, however.

Warning

Because rectangles rely on character offsets, rather than absolute screen positions, they may behave strangely when proportional-width fonts or tabs are used. Try to use monospaced fonts and soft tabs when editing columnar files.
