Chapter 9. Using Plugins

Dependency Errors

If jEdit detects that some plugin requires some capability that is not currently available, the plugin will not be loaded, and instead a "dependency error" will be printed in the activity log (Utilities>Show Activity Log) and system console. 

[error] JARClassLoader: ConsolePlugin requires jEdit 2.2final
[error] JARClassLoader: ErrorListPlugin requires class EditBusPlugin
[error] JARClassLoader: XInsertPlugin requires JDK 1.2

If a plugin you installed doesn't appear to be working, check the activity log for dependency errors.
