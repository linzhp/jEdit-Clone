Chapter 4. Basic Text Editing

Undo and Redo

Everybody makes mistakes. Especially when editing text. That's when jEdit's undo feature is useful.

File>Undo (keyboard equivalent: Control-Z) will reverse the most recently performed action. For example, if some text was deleted, this will bring it back again.

File>Redo (keyboard equivalent: Control-R) will redo the most recently undone action. For example, if some text was deleted, invoking Undo will make the text come back again. Invoking Redo after it was undone will once again delete that text.

jEdit stores the last 100 editing actions. An editing action can be an insertion, a deletion, or something more complicated, such as a Replace All operation.
