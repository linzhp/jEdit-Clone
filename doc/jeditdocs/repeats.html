Chapter 4. Advanced Features

Repeating Commands

Quite often while editing text, you need to insert a character or perform a command several times in a row. jEdit's repeat feature facilitates this. Keyboard commands, menu item selections and tool bar clicks can be repeated.

Pressing Control-Enter allows you to enter a number using the digit keys (0-9). The first non-digit character will be repeated that number of times. For example, "Control-Enter 1 4 Control-D" will delete 14 lines, and "Control-Enter 8 #" will insert the "#" character 8 times. While repeating is active, the repeat count is displayed in the view's status bar, instead of the usual caret position information.

Most commands behave as expected when repeated, with one exception: if given a repeat count, the Edit>Lines>Go to Line command will move the caret to that line. If no repeat count is given, it will display the go to line dialog box.
