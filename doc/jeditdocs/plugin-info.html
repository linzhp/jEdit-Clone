Chapter 10. Developing Plugins

Plugin Documentation

Unless your plugin does something extremely trivial, providing documentation is probably a good idea. For your plugin to appear in jEdit's help index, you must define the following properties, where plugin is the class name of your EditPlugin implementation:

  • plugin.plugin.name - the name of the plugin

  • plugin.plugin.author - the plugin author's name

  • plugin.plugin.version - the plugin version (this can be in any format you like, it is only used in the help system)

  • plugin.plugin.docs - the filename, relative to the plugin's JAR root, where the documentation is stored in HTML format

All these properties must be defined for your plugin to be listed in the help system. Here is an example: 

plugin.ConsolePlugin.name=Console
plugin.ConsolePlugin.author=Slava Pestov
plugin.ConsolePlugin.version=1.9.3
plugin.ConsolePlugin.docs=index.html
