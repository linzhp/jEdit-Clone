Chapter 5. Advanced Text Editing

Miscellaneous Commands

Over time, jEdit has accumulated a number of miscellaneous commands that defy categorization, yet are useful in some circumstances. This section will describe them.

Word Count

Edit>Word Count displays a dialog box with the number of characters, words and lines in the current buffer. A word is a string of text delimited by spaces.

Figure 5-6. The word count dialog box

Word Competion

Edit>Complete Word (keyboard equivalent: Control-B) attempts to compete the current word by searching the buffer for other words beginning with the current word. Invoking it multiple times will cycle through all possible completions. This can be used to enter previously typed, long identifiers in program source, for example.

A word is defined as a string of alphanumeric characters, and any non-alphanumeric characters listed in the "noWordSep" buffer-local property. For example, to be able to complete words containing hyphens ("-"), place the following in the first 10 lines of the buffer: 

:noWordSep=-:
