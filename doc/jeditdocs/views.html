Chapter 5. Advanced Text Editing

Working With Views

By default, jEdit only creates one editor window, or view. However, it is possible to have any number of views open at once, with each buffer being edited in one or more views. Views can also be split into multiple panes, for side-by-side viewing of two locations in a single buffer.

Utilities>New View (keyboard equivalent: Control-E Control-N) will create a new view. It will display the same buffer as the current view. A different buffer can also be selected from the Buffers menu.

Utilities>Close View (keyboard equivalent: Control-E Control-N) will close the current view. jEdit will exit if all views are closed.

Utilities>Split Horizonally (keyboard equivalent: Control-E 2) will horizontally split the text area into two regions. Each region can view a different location in the current buffer.

Utilities>Split Vertically (keyboard equivalent: Control-E 3) will vertically split the text are into two regions. Each region can view a different location in the current buffer.

Utilities>Unsplit (keyboard equivalent: Control-E 1) will remove all text areas areas except for the one which has keyboard focus from the view.

A single view can still only display one buffer at a time, even if it is split. Another limitation is that views are unsplit when some operations are performed; for example, switching buffers, reloading, and a few others.
