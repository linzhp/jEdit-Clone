Chapter 5. Advanced Text Editing

Working With Lines

Edit>Lines>Go to Line (keyboard equivalent: Control-L) moves the caret to the beginning of a line you specify.

An entire line can be selected by triple-clicking with the mouse. Another way to select a line is with the Edit>Selection>Select Line command (keyboard equivalent: Control-E W).

Edit>Lines>Select Line Range (keyboard equivalent: Control-E Control-L) prompts for a start line and end line, and selects all text between these lines, inclusive.

Edit>Lines>Scroll to Current Line (keyboard equivalent: Control-E Control-J) centers the current line on the screen.

Edit>Lines>Join Lines (keyboard equivalent: Control-J) removes any whitespace from the start of the next line and joins the current and next lines. This will beep if the caret is on the last line of the buffer. For example, if the caret is on the first line of the following example Java code: 

new Widget(Foo
       .createDefaultFoo());

Join Lines will change it to: 

new Widget(Foo.createDefaultFoo());

Edit>Lines>Delete Line (keyboard equivalent: Control-D) will delete the line that the caret is positioned on.

Edit>Lines>Delete to Start Of Line (keyboard equivalent: Control-Shift-Backspace) will delete all text from the caret position to the start of the line.

Edit>Lines>Delete to End Of Line (keyboard equivalent: Control-Shift-Delete) will delete all text from the caret position to the end of the line.
