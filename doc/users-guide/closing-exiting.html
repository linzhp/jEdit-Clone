Chapter 2. Working With Files

Closing Files and Exiting jEdit

File>Close Buffer (keyboard equivalent: Control-W) closes the current buffer. If it has unsaved changes, jEdit will ask if they should be saved first.

File>Close All Buffers (keyboard equivalent: Control-E Control-W) closes all buffers. If any buffers have unsaved changes, they will be listed in a dialog box where you can choose to save or discard them. Multiple buffers to operate on can be selected by clicking on them while holding down Control.

File>Exit (keyboard equivalent: Control-Q) will completely exit jEdit.
