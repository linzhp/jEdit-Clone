Chapter 7. Macros and Scripting

Recording Macros

Macros>Record Macro (shortcut: Control-M Control-R) prompts for a macro name and begins recording. While a recording is in progress, the string "(macro recording)" is displayed in the menu bar. jEdit records key strokes, commands invoked from the menu bar, tool bar clicks, and search and replace operations. Mouse clicks in the text area are not recorded; use the text selection commands or arrow keys when recording macros instead.

Macros>Stop Recording (shortcut: Control-M Control-S) stops recording. It also switches to the buffer containing the recorded macro, giving you a chance to check over the recorded commands and make any necessary changes. When you are happy with the macro, save the buffer and it will appear in the Macros menu. To discard the macro, close the buffer without saving.
