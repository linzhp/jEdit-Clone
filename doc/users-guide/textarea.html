Chapter 1. Basic Concepts

The Text Area

The text area is where text editing takes place. It behaves in a similar manner to many Windows and MacOS editors; the few unique features will be described in this section.

The text area will automatically scroll up or down if the caret is moved closer than three lines to the first or last visible line. This feature is called electric scrolling and can be disabled in the Text Area pane of the Utilities>Global Options dialog box.

To aid in locating the caret, the current line is drawn with a slightly different background color. With most editors, if a line of text ends with whitespace, it is not immediately apparent where the whitespace ends. jEdit remidies this by displaying end of line markers at the end of each line. End of line markers can be disabled in the Text Area pane of the Utilities>Global Options dialog box.

The vertical blue line is a wrap guide; entering characters beyond this position inserts a line break at the appropriate position. See the section called Inserting and Deleting Text in Chapter 3 for more information about word wrap.

Clicking the text area with the right mouse button displays a popup menu. Both this menu and the tool bar at the top of the view offer quick mouse-based access to frequently-used commands. The contents of the tool bar and context menu can be changed in the Utilities>Global Options dialog box.

The column and line number containing the caret, as well as the total number of lines in the buffer, is displayed in the bottom-right corner of the text area. In addition, the text area can optionally show line numbers next to displayed lines. To enable this, either double-click on the thin vertical strip to the left of the text area, or invoke View>Gutter Expanded (shortcut: Control-E Control-T). In addition to line numbers, the gutter also indicates which lines have markers or position registers pointing to them. (See the section called Markers in Chapter 3 and the section called Position Registers in Chapter 3)

The gutter can be resized by dragging with the mouse. The View>Save Gutter Size command makes the gutter's current size the default for future sessions.
