Creating New Files

If an untitled buffer is already open, it can be saved to create a new file. An untitled buffer can be created using the File>New File command (keyboard equivalent: Control-N). An alternate way to create a new file is to specify a non-existent file name when starting jEdit from your operating system's command line.
