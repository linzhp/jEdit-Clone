Keyboard Shortcuts

Many jEdit commands can be invoked using keystrokes. This speeds up editing by reducing the number of times you have to move your hands between the keyboard and mouse. Not all commands with keyboard shortcuts are accessable with one key stroke; for example, the keyboard shortcut for Scroll to Current Line is Control-E Control-J. That is, you must first press Control-E, followed by Control-J.
