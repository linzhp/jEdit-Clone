This chapter overviews the conventions used in this manual to display menu items and key bindings.

Menu Items

Because jEdit's menu hierarchy is rather complicated at times, the following convention is used to present menu items. First of all, the top level menu is listed, followed by successive levels of submenus, followed by the menu item itself. All menu components are separated by greater-than symbols (">"). For example, Edit>Lines>Indent Selected Lines refers to the Indent Selected Lines command of the Lines submenu of the Edit menu.

Menu items that end with an ellipsis (...) display a dialog box, and usually provide an option to cancel. Other menu items take effect immediately (but they may possibly show alerts or errors).
