Chapter 2. Working With Files

Opening Files

File>Open File (keyboard equivalent: Control-O) displays up a dialog box where the file to be edited can be located and chosen. For a detailed treatment of this dialog box, see the section called The File System Browser. The File>Recent Files menu lists the most recently opened files. The File>Current Directory menu lists all files in the current buffer's directory.

File>Insert File is similar to Open File, except that it inserts the contents of the specified file into the current buffer, instead of creating a new buffer.

Files that you do not have write access to are opened in read-only mode, and no text editing will be allowed to take place.
