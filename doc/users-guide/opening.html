Chapter 2. Working With Files

Opening Files

File>Open File (shortcut: Control-O) displays a file selection dialog box where the file to be edited can be located and chosen. The File>Current Directory menu lists all files in the current buffer's directory. File>Insert File displays a file selection dialog box and inserts the specified file into the current buffer. For a detailed treatment of the file chooser dialog box, see the section called The File System Browser.

jEdit remembers what files were opened most recently, along with where the caret was positioned within them. Recent files are listed in the File>Recent Files menu. When a recent file is opened, the caret is automatically moved to its location when the file was last edited. The number of recent files to remember can be changed and caret position saving can be disabled in the General pane of the Utilities>Global Options dialog box.

Files that you do not have write access to are opened in read-only mode, and making changes will not be allowed.
