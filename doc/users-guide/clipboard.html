Chapter 3. Editing Text

The Clipboard

Edit>Cut (shortcut: Control-X) and Edit>Copy (shortcut: Control-C) place the selected text in the clipboard. The two differ in that Cut then removes the selection from the buffer, while Copy leaves it in place.

File>Paste (shortcut: Control-V) will insert the clipboard contents in place of the selection (or at the caret position, if there is no selection). Edit>Paste Previous (shortcut: Control-E Control-V) displays a dialog box listing recently copied and pasted text. By default, the last 25 strings are remembered; this can be changed in the General pane of the Utilities>Global Options dialog box.

In the X Window System, there are actually two "clipboards"; the so called "primary selection", and the "Motif clipboard". jEdit only uses the Motif clipboard. However, the XClipboard plugin (see Chapter 8 for information about installing and using plugins) allows read-only access to the primary selection.
