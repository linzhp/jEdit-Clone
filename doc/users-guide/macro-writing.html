Chapter 7. Macros and Scripting

Evaluating BeanShell Expressions

Utilities>Evaluate prompts for a BeanShell expression, evaluates it, and displays the result in a dialog box. Utilities>Evaluate Selection evaluates the currently selected text, replacing it with the result. The possible uses for these two commands are endless; for example, they can be used to perform calculations. Try evaluating the following: 

Math.cos(Math.PI / 4) - 1 / Math.sqrt(2)
