Chapter 9. Installing and Using Plugins

Using Plugins

Exactly how a plugin can be used of course depends on the plugin. The first place to look in should probably be the plugin's help; plugins with documentation are listed in the Help menu. Some plugins might not have any documentation, however.

Most plugins add submenus or menu items to the Plugins menu. Some plugins add option panes to the Utilities>Global Options dialog box.
