Chapter 3. Editing Text

Working With Lines

An entire line can be selected by triple-clicking with the mouse, or using the Edit>Selection>Select Line command (shortcut: Control-E L). A selection that begins and ends on line boundaries can be created by triple-clicking and dragging.

Edit>Go to Line (shortcut: Control-L) prompts for a line number to move the caret to. Edit>Select Line Range (shortcut Control-E Control-L) selects all text between between two specified lines, inclusive.

Edit>Lines>Join Lines (shortcut: Control-J) removes any whitespace from the start of the next line and joins it with the current line. For example, invoking Join Lines on the first line of the following Java code: 

new Widget(Foo
       .createDefaultFoo());

Changes it to: 

new Widget(Foo.createDefaultFoo());

Edit>Lines>Delete Line (shortcut: Control-D) deletes the current line. Edit>Lines>Delete to Start Of Line (shortcut: Shift-Backspace) and Edit>Lines>Delete to End Of Line (shortcut: Shift-Delete) delete all text from the caret to the start and end of the line, respectively.

Edit>Lines>Delete Line (shortcut: Control-E R) removes all whitespace from the end of each selected line, or the current line if there is no selection.
