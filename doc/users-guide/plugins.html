jEdit supports plugins. Plugins are loadable modules of code that extend the editor's functionality. This chapter covers installation and use of plugins.

The Plugin Manager

Plugins>Plugin Manager will display the plugin manager window. It shows a list of all installed plugins. Clicking on a plugin will display information about it. To remove some plugins, select them and click Remove Selected Plugins. This will issue a confirmation dialog box first.
