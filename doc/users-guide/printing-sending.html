Chapter 3. Basic Features

Printing and Sending Files

File>Print (keyboard equivalent: Control-P) will print the current buffer. The printed output will have syntax highlighting. Each page will have a header with the file name, and a footer with the current date/time and page number.

File>Send will display the send file dialog box. "SMTP Server" should be set to your outgoing mail server. "From" and "To" are self-explanatory. "Subject" is set to the file name by default; this should be changed to something a little more descriptive. If Send selected text only is selected, only the selected text is sent; this is useful for e-mailing code snippets.
