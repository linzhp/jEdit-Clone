Chapter 1. Basic Concepts

Command Repetition

To repeat a command any number of times, invoke Utilities>Repeat Next Command (keyboard equivalent: Control-Enter) and enter the desired repeat count, followed by the command to repeat (either a keystroke or menu item selection). For example, "Control-Enter 1 4 Control-D" will delete 14 lines, and "Control-Enter 8 #" will insert "########" in the buffer.
