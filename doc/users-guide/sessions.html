Chapter 2. Working With Files

Editor Sessions

When working on a project, it can be convenient to save a list of open files, and re-open those files at a later time. jEdit's saved sessions do exactly that.

Sessions are plain text files with the extension .session, stored in the .jedit/sessions subdirectory of the user's home directory. The File>Load Session and File>Save Session commands load and save sessions.

The session named default.session is loaded on startup, and the list of open files is recorded into it when jEdit exits. The end result is that open files are retained if jEdit is closed and reopened. This default session feature can be disabled in the Editor pane of the Utilities>Global Options.
