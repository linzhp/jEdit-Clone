Chapter 1. Basic Concepts

Views

Each editor window is referred to as a view. It is possible to have multiple views open at once. View>New View creates a new view. View>Close View closes the current view. If only one view is open, closing it will exit jEdit, unless background mode is on; see Appendix D for more information about background mode.

A quicker way to edit two locations simultaneously is to split the view. View>Split Horizontally (keyboard equivalent: Control-2) splits the text area into two text areas, stacked above each other. View>Split Vertically (keyboard equivalent: Control-3) splits the text area into two text areas, next to each other. View>Unsplit (keyboard equivalent: Control-1) removes all but the current text area from the view. Editing commands operate in the text area that has keyboard focus (or if no text area is focused, the one that last had keyboard focus).
