Chapter 3. Basic Features

Creating New Files

If an "Untitled" buffer is already open, you can just begin typing and save it to create a new file. You can also create a new untitled buffer by invoking File>New File (keyboard equivalent: Control-N). An alternate way to create a new file is to specify a non-existent file name when starting jEdit.
