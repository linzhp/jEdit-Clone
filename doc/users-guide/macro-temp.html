Chapter 7. Macros and Scripting

Temporary Macros

Sometimes, a complicated operation only needs to be repeated a few of times, and it is not worth creating a new macro file. The temporary macro feature is useful in those situations.

Macros>Record Temporary Macro (shortcut: Control-M Control-M) begins recording to a buffer named Temporary_Macro.bsh.

Macros>Stop Recording (shortcut: Control-M Control-S) stops recording the temporary macro. This is the same command used to stop recording an ordinary macro. Once recording is complete, you don't need to save the Temporary_Macro.bsh buffer before playing it back- but if you do, it will become an ordinary macro, displayed in the Macros menu.

Macros>Play Temporary Macro (shortcut: Control-M Control-P) plays the macro recorded to the Temporary_Macro.bsh buffer.
