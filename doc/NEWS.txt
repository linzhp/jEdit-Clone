OVERVIEW OF CHANGES BETWEEN JEDIT 3.1 AND 3.2

- Improved markers
- Multiple selection
- Middle mouse button
- BeanShell search and replace
- Cut Append, Copy Append, Cut Append to Register commands
- Search in Open Buffers command, Search in Directory command
- Search in Directory in file system browser right-click menu
- Reload Edit Modes command
- Verilog syntax highlighting
- Status bar
- Improved plugin manager
