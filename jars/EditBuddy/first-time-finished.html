Further information

Thank you for completing the initial setup wizard!

The jEdit user guide, along with plugin documentation, can be viewed from the Help menu. If you would like to print the user guide, A PDF (Adobe Acrobat) version is available as an optional download from http://jedit.sourceforge.net/download.php.

If you would like to keep up-to-date in the world of jEdit, subscribe to one of the mailing lists. More information is available from the jEdit web site, located at http://jedit.sourceforge.net.

Now click the button labelled Finish below and have fun using jEdit!
