Updating plugins

Plugins are very easy to update; the plugin manager feature can obtain a list of updated plugins, and then download and install any that interest you.

To update your plugins now, make sure you are connected to the Internet and click the button below. Note that plugins you update now will not be usable immediately; you must restart jEdit first.

You may update plugins at any other time by invoking the Plugins>Plugin Manager command from jEdit's menu bar. If for some reason you have problems updating plugins using the plugin manager, you can download and install them manually. Point your WWW browser to http://jedit.standmed.com for more information.
