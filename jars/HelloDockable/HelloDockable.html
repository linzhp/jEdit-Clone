The wonderful HelloDockable plugin

This plugin demonstrates the jEdit 2.6 dockable window API, and the jEdit 2.7 action API.

It implements a simple dockable window that displays a text area. The contents of the text area are made persistent using properties.

To specify where the window is to be docked, see Utilities->Global Options->Docking.
