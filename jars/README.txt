SAMPLE PLUGINS

The jEdit source distribution includes source code for three sample
plugins:

- HelloWorld: demonstrates adding menu items, displaying canned dialog
  boxes

- HelloEditBus: demonstrates listening for editor events, using the
  ErrorSource API for in-place error highlighting. Requires EditBus
  plugin.

- HelloDockable:  demonstrates adding menu items, displaying dockable
  windows, using jEdit properties for persistance.
